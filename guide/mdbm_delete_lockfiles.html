mdbm_delete_lockfiles¶

SYNOPSIS¶

mdbm_delete_lockfiles [-h] mdbm

DESCRIPTION¶

mdbm_delete_lockfiles deletes all the lockfiles of a specified MDBM. It requires the fully qualified path to an MDBM. USE THIS UTILITY WITH EXTREME CAUTION! Use only after you are completely done with the MDBM and no other processes are using it. Otherwise, the MDBM will likely become corrupted.

OPTIONS¶

-h Show help message

RETURN VALUE¶

Returns 0 upon success, 1 upon failure.

EXAMPLES¶

mdbm_delete_lockfiles /tmp/foo.mdbm
